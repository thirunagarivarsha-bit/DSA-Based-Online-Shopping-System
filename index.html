🛒 SmartShop - DSA Based Online Shopping System

A menu-driven Java project demonstrating real-world usage of Data Structures and Algorithms.

📚 DSA Concepts Used

  • ArrayList → Product Storage
  • LinkedList → Cart Management
  • Stack → Undo/Redo Operations
  • Queue → Order Processing
  • HashMap → Fast Product Search

🔍 Searching Algorithms

  • Linear Search (Search by ID)
  • Binary Search (Search by Name)
  • Hashing (Fast Lookup)

🔄 Sorting Algorithms

  • Quick Sort → Price Ascending
  • Merge Sort → Price Descending
  • Selection Sort → Name
  • Insertion Sort → ID

✨ Features

  • Add / Remove Products from Cart
  • Undo / Redo using Stack
  • Queue-based Order Processing
  • File Handling for Product Data
  • Low Stock Alert System
  • Discount & Festival Offer System
  • Billing System

🚀 Innovative Features

  • ⚠ Low Stock Detection
  • 🎉 Festival Offer (Auto Apply)
  • 💰 Smart Discount System
  • 📊 Most Expensive Product Finder
  • 🧠 Smart Product Recommendation
  • 📉 Price Filter Option

# DSA-Based-Online-Shopping-System
A menu-driven Online Shopping and Inventory Management System built using core Data Structures in Java. Implements LinkedList (cart), Stack (undo/redo), Queue (order processing), Hashing (search), Sorting and File Handling with real-world features like stock alerts, discount system, and order billing.
Features
Product management using ArrayList
Cart using LinkedList
Undo/Redo using Stack
Order processing using Queue
Searching using Hashing
Sorting (Quick, Merge, etc.)
Technologies
Java
Data Structures
How to Run
Open in Eclipse
Run Main.java
